# 剖析

百度百科解释：

    二叉树（Binary tree）是树形结构的一个重要类型。
    许多实际问题抽象出来的数据结构往往是二叉树形式，即使是一般的树也能简单地转换为二叉树，
    而且二叉树的存储结构及其算法都较为简单，因此二叉树显得特别重要。
    二叉树特点是每个结点最多只能有两棵子树，且有左右之分 [1]  。
    二叉树是n个有限元素的集合，该集合或者为空、或者由一个称为根（root）的元素及两个不相交的、
    被分别称为左子树和右子树的二叉树组成，是有序树。
    当集合为空时，称该二叉树为空二叉树。
    在二叉树中，一个元素也称作一个结点
